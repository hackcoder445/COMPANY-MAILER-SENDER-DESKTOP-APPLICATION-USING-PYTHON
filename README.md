# COMPANY-MAILER-SENDER-DESKTOP-APPLICATION-USING-PYTHON
Professional desktop mailer for automated bulk email campaigns. Built with Python and PyQt6 for speed, reliability, and ease of use.

## Features
- SMTP account management
- Bulk email sending
- Templates management
- Contacts import
- Send history

## Setup
```bash
python -m venv .venv
.\.venv\Scripts\activate
pip install -r requirements.txt
python main.py
```
